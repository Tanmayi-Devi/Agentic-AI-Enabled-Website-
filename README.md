# Finance Tracker

A modern, dark-themed personal finance management application that helps users track their income, expenses, and monthly budget goals.

## Features
* **Dashboard:** Get a quick overview of your total balance, income, and expenses.
* **Transaction Management:** Easily add new transactions and view your complete history.
* **AI Assistant:** Get personalized financial insights using the integrated Google Gemini AI.
* **Custom UI:** A sleek "Space/Purple" themed interface for a premium user experience.

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** Node.js, Express.js
* **AI Integration:** Google Generative AI SDK
* **Data Storage:** Browser LocalStorage

## Project Structure
* `/public`: Contains all frontend `.html` files.
* `/public/css`: Contains `style.css` for application styling.
* `server.js`: The Express server handling API requests.
* `package.json`: Project configuration and dependencies.
